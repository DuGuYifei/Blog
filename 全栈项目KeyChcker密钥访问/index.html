全栈项目KeyChcker密钥访问

Posted on 2023-02-28 In 全栈
Prepare a "coat" for other programs which user can use a time-limited key to log in. User's machine is bound. Supervisor have has his own interface. Use SHA-256 encrytion, http request, Multi-processes communication(Shared Memory, Message Pipe), Multithread, WinAPI.

Intro

Github link

Prepare a “coat” for other programs which user can use a time-limited key to log in. Their machines are bound.

The wrapper framework use mysql and contains back-end (Springboot), back-end supervise interface (vue) and user client (Qt c++).

Also display a wrappee example by pyhon to use the shared memory or message pipe.

Tech stack

  1. Springboot - java
  2. Vue.js - javascript, html, css
  3. Qt desktop - C++
  4. Python script
  5. Database - MySQL
  • Http request
  • WinAPI - C++, python
  • Multi-processes communication - Shared Memory, Message Pipe
  • Multithread in Qt C++
  • Test by Docker
